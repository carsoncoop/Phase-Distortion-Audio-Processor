# Phase Distortion Audio Effect Plugin

A VST3 audio effect plugin built with JUCE and C++ that applies phase distortion processing to incoming audio signals in real time.

## Overview

This project explores digital signal processing concepts through the development of a custom audio effects plugin. The plugin is designed to manipulate the phase characteristics of incoming audio to create unique harmonic and timbral changes.

## Technologies

- C++
- JUCE Framework
- CMake
- VST3

## Current Features

- Audio output processing
- Basic plugin architecture using JUCE
- Parameter management framework
- Frequency modulation
- Basic user interface

## Planned Features

- Adjustable distortion amount
- Preset saving/loading
- Additional phase shaping modes
- Real-time waveform visualization
- Performance optimization

## Goals

- Learn real-time audio programming
- Develop stronger C++ skills
- Explore digital signal processing techniques
- Build a professional-quality audio plugin

1. Clone the repository
2. Open the project in CLion or your preferred IDE
3. Configure CMake
4. Run configuration
